# 《Rebirth of the Legendary Sword Master》完整大纲

## 主角设定
- **姓名**: 陈凡 (Chen Fan)
- **前世身份**: 天剑宗第一剑修，被徒弟背叛身亡
- **重生身份**: 十四岁入门弟子，带着前世记忆
- **金手指**: 前世修炼经验和武技记忆
- **目标**: 复仇 + 修炼到极致 + 避免前世错误

## 主要人物
1. **陈凡** - 主角，重生者
2. **张伟** - 前世背叛的徒弟，今世仍是同门
3. **林雪** - 前世背叛的女人（宗主之女），今世还未相遇
4. **天剑宗宗主** - 林雪之父，强大但偏心
5. **李长老** - 外门教习，起初看不起陈凡后来刮目相看
6. **王铁山** - 外门弟子，憨厚老实，成为陈凡第一个朋友
7. **周天明** - 内门弟子，天才型，竞争对手
8. **神秘老者** - 隐藏boss，前世陈凡未发现的重要人物

## 世界观设定
- **修为等级**: 炼体境 → 凝气境 → 筑基境 → 金丹境 → 元婴境 → 化神境 → 大乘境 → 渡劫境
- **势力分布**: 天剑宗（正道领袖）、万魔宗（魔道）、皇室、各大家族
- **特殊元素**: 剑意、剑魂、天地灵气

## 核心剧情线
1. **成长线**: 从外门弟子到内门首席
2. **复仇线**: 逐步揭露前世背叛真相
3. **感情线**: 与林雪的纠缠（前世她背叛他）
4. **阴谋线**: 揭示宗门内部更深层的阴谋
5. **传承线**: 获得上古剑道传承

## 第一部分：重生归来（第1-10章）

### 第一章：重生
**提示词**: 背叛、死亡、重生、童年、决心改变
**字数**: 约3500字
**内容**: 陈凡被张伟和林雪联手背叛，在禁地密室中重伤垂死。临死前回忆起一生的遗憾，决定若有来世绝不重蹈覆辙。突然惊醒发现自己重生到十四岁刚入门那天，激动不已。

**提示词**: 熟悉环境、旧友重逢、外门弟子、差距感

### 第三章：惊人天赋
**提示词**: 突破常规、震惊教习、展现实力、引起注意
**字数**: 约3500字
**内容**: 在外门训练中，陈凡用前世记忆中的技巧展现惊人实力，震惊李长老和张伟。张伟既惊讶又有些嫉妒。王铁山开始接近陈凡。

**字数**: 约3400字
**内容**: 陈凡深夜在后山偶遇一位神秘老者（隐藏boss的线索），老者看出了陈凡的不凡，给了他一些指点。这是前世没有发生的事，陈凡意识到重生后很多事情正在改变。

**字数**: 约3500字

### 第八章：前世功法
**提示词**: 天剑心法、修炼加速、实力飞跃、引起怀疑
**字数**: 约3400字
**内容**: 陈凡开始修炼前世掌握的天剑心法，进境神速。他的实力提升速度引起了周天明的怀疑，后者开始暗中调查他。

### 第九章：林雪示好
**提示词**: 追求、拒绝、内心挣扎、前世阴影
**字数**: 约3200字
**内容**: 林雪主动接近陈凡，表达好感。但前世的记忆让陈凡对她充满戒备。他刻意保持距离，却又忍不住想起前世的点点滴滴。

## 写作技巧提示

### 每章必备元素
1. **冲突与悬念**: 每章结尾留下悬念
2. **角色成长**: 主角或配角有新的成长
3. **情感渲染**: 适当的感情戏
4. **战斗描写**: 精彩的动作场面
5. **世界观丰富**: 不断揭示新的设定

### 爽点设置
1. **打脸时刻**: 反派嚣张后被打脸
2. **扮猪吃虎**: 隐藏实力后震惊全场
3. **英雄救美**: 关键时刻拯救美人
4. **逆袭翻盘**: 以弱胜强的快感
5. **实力飙升**: 突破境界的爽感

### 虐点设置
1. **悲剧英雄**: 战友牺牲
2. **命运捉弄**: 相爱的人不能在一起
3. **艰难抉择**: 两难的选择
4. **往事追忆**: 痛苦回忆的折磨
5. **孤独旅程**: 无人理解的寂寞
